有名絵画クイズ

世界の名画について楽しく学べるアートクイズアプリ

有名絵画クイズは、ルネサンスから現代アートまで、幅広い時代の名画について学べる教育アプリです。作品名、画家、制作年代、技法など、多角的な視点から美術作品への理解を深められます。

美術館に行く前の予習や、アート鑑賞の知識を深めたい方に最適。クイズを楽しみながら、自然と美術史の知識が身につきます。

スクリーンショット

機能と特徴

  • 1000作品以上の名画を収録
  • 時代別・画家別・ジャンル別の学習モード
  • 作品の背景や技法の詳細解説
  • 高解像度画像でディテールまで鑑賞
  • 美術館情報と連携した作品紹介
  • お気に入り作品のコレクション機能
  • 学習の進捗管理とランキング機能
  • 定期的な新作品の追加更新

技術情報

開発での工夫点

美術作品の魅力を最大限に伝えるため、高解像度画像の効率的な表示システムを構築しました。React Native Fast Imageを活用したプログレッシブローディングとキャッシングにより、快適な鑑賞体験を提供しています。

また、美術史の専門家監修のもと、教育的価値の高いコンテンツを制作。単なるクイズアプリを超えた、本格的な美術学習ツールとして設計しました。

使用技術

  • React Native
  • Expo
  • AsyncStorage
  • React Native Fast Image
  • Firebase
  • React Native WebGL
