激ムズ！フォントクイズ

デザイナー必見！フォントの知識を試すチャレンジングなクイズ

激ムズ！フォントクイズは、タイポグラフィーに関する知識を試す本格的なクイズアプリです。プロのデザイナーも唸る難問から、初心者でも楽しめる基本問題まで幅広くカバー。

フォントの歴史、特徴、使用例など、タイポグラフィーの奥深い世界を楽しみながら学べます。デザインの現場で役立つ実践的な知識も身につきます。

スクリーンショット

機能と特徴

  • 500種類以上のフォントを収録
  • フォント識別・比較・歴史など多様な問題形式
  • 詳細なフォント解説と使用例の紹介
  • 難易度別のステージ制クイズ
  • 全国ランキングで腕試し
  • お気に入りフォントの保存機能
  • フォントの特徴を学べる学習モード
  • 定期的な新問題の追加

技術情報

開発での工夫点

このアプリの開発では、フォントの微妙な違いを正確に表示することに特にこだわりました。カスタムフォントの動的読み込みシステムを構築し、高品質なフォントレンダリングを実現しています。

また、問題の難易度調整には機械学習を活用。ユーザーの正答率を分析し、個々のレベルに合わせた問題を出題する仕組みを実装しました。

使用技術

  • Swift
  • SwiftUI
  • Core Text
  • Core ML
  • CloudKit
  • GameKit
