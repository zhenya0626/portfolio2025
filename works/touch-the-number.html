Touch the Number

数字を順番にタッチする反射神経トレーニングゲーム

Touch the Numberは、画面に表示される数字を順番にタッチしていくシンプルなゲームです。簡単なルールながら、集中力と反応速度を同時に鍛えることができます。

スコアアタックの要素もあり、自分の記録を更新していく楽しさや、友達と競い合う要素も充実。年齢を問わず楽しめるカジュアルゲームとして人気を博しています。

スクリーンショット

機能と特徴

  • 直感的で分かりやすいゲームシステム
  • 複数の難易度レベル（3×3から6×6グリッド）
  • 精密なタイム測定とスコア計算
  • 詳細な統計とパフォーマンス分析
  • リーダーボードで全国ランキング
  • 練習モードで気軽にプレイ
  • カラフルで見やすいデザイン
  • サウンドエフェクトとハプティクス

技術情報

開発での工夫点

ゲーム性を重視し、ミリ秒単位での正確なタイミング測定を実現。Core Animationを活用したスムーズなアニメーションと、ハプティクスフィードバックにより、プレイヤーに心地よい操作感を提供しています。

また、GameKitを活用したリーダーボード機能により、世界中のプレイヤーと競い合える環境を構築。モチベーション維持に役立つ要素を多数盛り込みました。

使用技術

  • Swift
  • SwiftUI
  • Core Animation
  • GameKit
  • AVFoundation
  • Core Haptics
